1) make sure to install php5 with pear enabled, i.e.

    sudo port install php5 +apache2+darwin_8+macosx+mysql5+postgresql+pear+sqlite

2) tell pear about where php.ini lives

    sudo pear config-set php_ini /opt/local/etc/php.ini

3) upgrade pear to the latest version of itself

    sudo pear upgrade-all

4) upgrade pear to know about protocol upgrades

    sudo pear channel-update pear.php.net
    sudo pear channel-update pecl.php.net

5) install the APC caching extension

    sudo pear install APC
